Домашняя работа выполнялась одним человеком. В связи с нехваткой времени пришлось сильно упростить проект. Поэтому объясню конечную реализацию  системы.
Есть агент посетитель, агент управляющий и агент повар. Посетитель запрашивает у управляющего меню и говорит сколько времени готов потратить. 
Управляющий выдает ему часть меню, содержащую  блюда, которые успеют приготовиться. Посетитель выбирает что хочет заказать и говорит об этом управляющему. Управляющий передает это повару, который готовит блюда. 
Как только блюда приготовлены их можно забрать в окошке выдачи. Менеджер принимает меню, как json файл в формате 
{
            "menu_dishes": [{
                                    "menu_dish_name": pelmeni,
                                    "menu_dish_time": 20,
                                    "menu_dish_price": 59
                        }
            ]
}
Надеюсь наскребу хотя бы на 3/10 я старался)
